# E-Voting System – College Mini Project

## Project Description

The **E-Voting System** is a simple web-based application developed for college elections. It allows students to vote for their preferred candidates through a user-friendly interface. The system demonstrates the basic concept of online voting using front-end web technologies.

This project is designed as a **prototype of a digital voting system** where students can log in, select a candidate, and cast their vote. An admin panel is also included to view the voting results.

## Objectives

* To demonstrate how an online voting system works.
* To create a simple and aesthetic web interface for voting.
* To store and display voting results dynamically.
* To understand the use of HTML, CSS, and JavaScript in building interactive web applications.

## Technologies Used

* **HTML** – Structure of the website
* **CSS** – Styling and user interface design
* **JavaScript** – Voting logic and data handling
* **LocalStorage** – To store vote data in the browser

## Features

* Student login interface
* Candidate selection with images
* One-click voting system
* Admin login with password protection
* Vote result display in the admin panel
* Modern and responsive UI design

## How to Run the Project

1. Download or clone the project files.
2. Open the project folder.
3. Run **index.html** in any web browser.
4. Enter your name to access the voting page.
5. Select a candidate and submit your vote.
6. Admin can log in to view results.

## Limitations

* Votes are stored in **browser LocalStorage**, so they are saved only in the same browser.
* The project does not include a backend or database.
* This project is intended for **educational demonstration purposes only**.
